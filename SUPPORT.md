# Support

## Questions and Usage Help

- Open a GitHub issue for bug reports, documentation gaps, and feature requests.
- Use the README and `docs/onboarding.md` for local setup and contributor workflow.

## What This Public Repo Covers

This repository documents local development, testing, and contribution workflows.
Private deployment infrastructure and maintainer-only operations are intentionally
not part of the public support surface.

## Security Issues

Do not use public issues for security-sensitive reports. Follow `SECURITY.md`.
